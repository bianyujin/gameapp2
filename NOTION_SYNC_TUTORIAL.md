# 🚀 Notion同步配置完整教程

## 第一步：创建Integration（免费）

1. 访问 https://www.notion.so/my-integrations
2. 点击 **"+ New integration"**
3. 填写信息：
   - **Name**: `GameHub Sync`（随便取）
   - **Associated workspace**: 选择你的工作区
4. 点击 **Submit**
5. 复制 **Internal Integration Token**（以 `secret_` 开头）

```
示例：secret_1a2b3c4d5e6f7g8h9i0j...
```

**保存好这个Token，后面要用！**

---

## 第二步：连接你的Notion表格

### 如果你已有Notion表格：

1. 打开你的Notion表格页面
2. 点击右上角 **"..."** → **"Add connections"**
3. 搜索并选择你刚创建的 `GameHub Sync`
4. 点击 **Confirm**

### 获取数据库ID：

从浏览器地址栏复制：

```
https://www.notion.so/your-workspace/DATABASE_ID?v=xxx
                                    ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
                                    这就是数据库ID
```

示例：`1a2b3c4d5e6f7g8h9i0j1a2b3c4d5e6f`

---

## 第三步：配置表格属性

确保你的Notion表格有以下属性（列名要完全一致）：

| 属性名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 标题 | Title | ✅ | 游戏名称 |
| 图标 | Text | ❌ | emoji图标 |
| 类型 | Select | ❌ | 游戏类型 |
| 评分 | Number | ❌ | 评分(0-100) |
| 社团 | Text | ❌ | 制作社团 |
| 评价 | Text | ❌ | 简短评价 |
| 介绍 | Text | ❌ | 详细介绍 |
| 封面 | Text | ❌ | 封面图片URL |
| 隐藏备注 | Text | ❌ | 仅管理员可见 |
| 是否公开 | Checkbox | ❌ | 勾选=用户可见 |

**注意**：属性名可以用中文或英文，但要和代码中的字段名对应。

---

## 第四步：在应用中配置

1. 打开应用 → **个人中心**
2. 点击 **云端设置**
3. 选择 **Notion** 标签
4. 粘贴Token和数据库ID
5. 点击 **测试连接**
6. 保存设置

---

## 第五步：同步数据

### 自动同步
- 勾选"自动同步"
- 每次打开应用自动获取最新数据

### 手动同步
- 点击 **从云端获取** → 下载数据

---

## 🔒 敏感内容过滤

### 方案一：使用"是否公开"字段

在Notion中：
- 勾选"是否公开" → 用户可见
- 不勾选 → 仅管理员可见

### 方案二：使用"隐藏备注"字段

- 敏感信息写在"隐藏备注"列
- 应用会自动过滤，用户看不到

### 方案三：创建两个表格

```
表格1：公开数据（用户可见）
表格2：私密数据（仅你可见）
```

---

## 📊 数据同步逻辑

```
Notion表格
    ↓ API读取
应用获取数据
    ↓ 过滤处理
├── 是否公开=勾选 → 显示给用户
├── 是否公开=未勾选 → 仅管理员可见
└── 隐藏备注字段 → 自动过滤
```

## ❓ 常见问题

### Q: Token会泄露吗？
A: Token存储在本地，不会上传。打包APK后更安全。

### Q: 同步有延迟吗？
A: 通常几秒内完成。取决于网络状况。

### Q: 可以多个应用共用一个表格吗？
A: 可以！多个应用可以读取同一个Notion数据库。

### Q: 用户评论存在哪里？
A: 用户评论存储在本地或Firebase，不会写入Notion。
