# 数据管理指南

GameHub Pro 支持多种方式输入和管理数据，无需重新打包即可动态更新内容。

---

## 📥 数据输入方式

### 方式一：手动添加（推荐新手）

1. 点击应用右下角的 **+** 按钮
2. 填写游戏信息：
   - 游戏名称（必填）
   - 图标（emoji）
   - 分类
   - 评分
   - 下载量
   - 描述
3. 点击"添加"保存

### 方式二：JSON格式导入

1. 进入 **个人中心** → 点击 **导入数据**
2. 选择 **JSON格式** 标签
3. 粘贴JSON数据，格式如下：

```json
[
  {
    "title": "星际探险",
    "icon": "🚀",
    "category": "冒险",
    "rating": 4.8,
    "downloads": "250万+",
    "description": "一款开放世界太空冒险游戏"
  },
  {
    "title": "王国保卫战",
    "icon": "🏰",
    "category": "策略",
    "rating": 4.9,
    "downloads": "500万+",
    "description": "经典塔防游戏续作"
  }
]
```

4. 选择导入模式：
   - **追加到现有数据**：在现有数据基础上添加
   - **替换所有数据**：清除现有数据，导入新数据
5. 点击"导入"

### 方式三：CSV格式导入

1. 进入 **个人中心** → 点击 **导入数据**
2. 选择 **CSV格式** 标签
3. 粘贴CSV数据，格式如下：

```csv
游戏名称,图标,分类,评分,下载量,描述
星际探险,🚀,冒险,4.8,250万+,一款开放世界太空冒险游戏
王国保卫战,🏰,策略,4.9,500万+,经典塔防游戏续作
```

4. 选择导入模式
5. 点击"导入"

## ✏️ 数据编辑

### 编辑单条数据

1. 进入 **数据管理** 页面
2. 找到要编辑的游戏
3. 点击 **编辑** 按钮
4. 修改信息后点击 **保存**

### 删除数据

1. 进入 **数据管理** 页面
2. 找到要删除的游戏
3. 点击 **删除** 按钮
4. 确认删除

### 批量操作

1. 勾选要操作的游戏（可使用全选）
2. 进行批量删除等操作

---

## 📤 数据导出

### 导出为CSV

1. 进入 **数据管理** 页面
2. 可先筛选需要的数据
3. 点击工具栏的 **📤 导出** 按钮
4. CSV文件将自动下载

---

## 💾 数据存储说明

### 本地存储

- 数据存储在浏览器的 **localStorage** 中
- 关闭浏览器后数据不会丢失
- 清除浏览器数据会导致数据丢失

### 数据持久化

所有操作（添加、编辑、删除、导入）都会自动保存到本地存储。

---

## 🔄 数据重置

如需恢复默认数据：

1. 进入 **个人中心**
2. 点击 **清除缓存**
3. 确认清空数据
4. 应用将恢复到初始示例数据

## 📋 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | String | ✅ | 游戏名称 |
| icon | String | ❌ | 图标（推荐emoji） |
| category | String | ❌ | 分类（动作/角色扮演/策略/休闲/竞技/冒险） |
| rating | Number | ❌ | 评分（0-5） |
| downloads | String | ❌ | 下载量（如：100万+） |
| description | String | ❌ | 游戏描述 |

## 💡 使用技巧

### 快速批量导入

1. 先用Excel或Google Sheets整理数据
2. 导出为CSV格式
3. 复制CSV内容粘贴到导入框

### 图标推荐

使用emoji作为游戏图标，常用emoji：
- 🎮 游戏
- 🚀 太空/科幻
- 🏰 城堡/策略
- ⚔️ 动作/战斗
- 🧙 角色扮演
- 🏆 竞技
- 🎪 休闲
- 🗺️ 冒险

### 分类建议

- **动作**：战斗、射击类游戏
- **角色扮演**：RPG类游戏
- **策略**：塔防、策略类游戏
- **休闲**：消除、益智类游戏
- **竞技**：MOBA、FPS类游戏
- **冒险**：探索、剧情类游戏

## ❓ 常见问题

**Q: 导入后数据在哪里查看？**
A: 导入的数据会显示在"数据管理"和"首页"的游戏列表中。

**Q: 数据会同步到其他设备吗？**
A: 目前数据存储在本地，不会自动同步。可以通过导出/导入功能手动迁移数据。

**Q: 最多可以存储多少条数据？**
A: localStorage通常支持5-10MB，大约可以存储数千条游戏数据。

**Q: 如何备份数据？**
A: 定期使用"导出数据"功能导出CSV文件保存。

## 🚀 高级用法

### 通过代码直接修改

打开浏览器开发者工具（F12），在控制台中：

```javascript
// 查看所有数据
console.log(App.games);

// 添加新游戏
App.games.push({
    id: App.nextId++,
    title: "新游戏",
    icon: "🎮",
    category: "休闲",
    rating: 4.5,
    downloads: "50万+",
    description: "游戏描述",
    updateDate: new Date(),
    isFavorite: false
});
App.saveData();
App.render();

// 清空所有数据
App.games = [];
App.saveData();
App.render();
```

### 从外部API获取数据

可以编写脚本从游戏API获取数据，转换为JSON格式后导入应用。
