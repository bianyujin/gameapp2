# 应用图标目录

请在此目录下放置以下尺寸的PNG格式应用图标：

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

建议使用游戏手柄或GameHub相关的设计作为图标主题。

可以使用在线工具如 [favicon.io](https://favicon.io/) 或 [realfavicongenerator.net](https://realfavicongenerator.net/) 来生成这些图标。
