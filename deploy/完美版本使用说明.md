## 📋 完整流程

### 首次使用（仅需一次！）

1. **Notion 导出 CSV**
2. **打开 `一键发布工具.html`**
3. **填写配置（仅第一次！）：**
   - APP最新版本号
   - 管理员密码
   - games.json GitHub 下载地址（上传后填）
   - Notion嵌入地址（可选）
4. **选择 CSV，处理**
5. **下载 games.json 和 config.json**
6. **上传 GitHub Releases**
7. **APP连点版本5次 → 登录管理员**
8. **在管理员面板填 GitHub Config URL**
9. **点击同步 → 完成！**

---

### 日常更新（超级简单！）

1. **Notion 里更新数据 → 导出 CSV**
2. **打开 `一键发布工具.html`**
   - ✅ 所有配置自动加载（不用再填！）
   - ✅ 数据版本号自动递增（不用手动改！）
3. **选择新 CSV → 处理**
4. **检查字段可见性（可选）**
5. **下载 games.json 和 config.json**
6. **上传 GitHub Releases 替换旧文件**
7. **✅ 用户下次点击同步自动拿到新数据！**
